NET Pay Program

This program is a simply NET pay calculator. It takes your GROSS Pay and the dedecuts Federal, State, FICA. It has the option to deduct any additional amount from check. Then it displays your NET pay.

  1. START
  2. DELARATIONS:
    • FED = grossPay * .15
    • STATE = grossPay * .05
    • FICA = grossPay * .0762
    • additionalDeduction = 0
    • netPay = 0
  3. DISPLAY "Enter in your grossPay here"
  4. INPUT grossPay
  5. DISPLAY "We will deduct Federal, State, and FICA taxes from your NET pay."
  6. netPay = grossPay - (FED + STATE + FICA)
  7. DISPLAY "Would you like to make any additional deductions? yes/no"
  8. INPUT ans
  9. PROCESS if ans === true/yes
    1. DISPLAY "Enter additional amount"
    2. INPUT additionalDeduction
    3. PROCESS netPay = netPay - additionalDeduction
  10. DISPLAY "Here is your NET Pay 'netPay'."
  11. END
