PTO Program

PTO V1.0

A simple program that calculates the PTO(Paid time off) for 1yr at a fixed rate of 5hrs per paycheck paid bi-weekly.

STEPS

  1. Start
  2. SET INT paycheckPerYear
  3. SET INT hrsPerPayCheck
  4. Display "This is a simple program that outputs how many hrs you earn a year for PTO(Paid Time Off)."
  5. SET INT totalHrsPerYr = hrsPerPaycheck * paycheckPerYear
  6. Display "You earn totalHrsPerYr hours per year"
  7. End

PTO V2.0

A simple program that calculates the PTO(Paid time off) for the corresponding amount of time the employee has work in years. Returns amoun of PTO in weeks.

STEPS

  1. Start
  2. SET INT hrsPerPaycheck
  3. SET INT paycheckPerYear = 26
  4. Display "How many years have you worked"
  5. INPUT yearsWorked
  6. if yearsWorked less than 5 hrsPerPayCheck = 5
  7. if yearsWorked equal to or greater than 5 hrsPerPayCheck = 6
  8. if yearsWorked equal to or greater than 10 hrsPerPayCheck = 7
  9. SET INT totalHrsPerYr = hrsPerPayCheck * paycheckPerYear
  10. SET REAL weeks = totalHrsPerYr/40
  11. Display "You get 'weeks' weeks off a year"
  12. End
